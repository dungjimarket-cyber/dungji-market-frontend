## 📋 테스트 실행 방법

### 1. 전체 테스트 실행
```bash
npm test
```

### 2. 특정 테스트 파일 실행
```bash
npm test issue-test
```

### 3. 커버리지 리포트 생성
```bash
npm test -- --coverage
```

### 4. Watch 모드로 실행 (파일 변경시 자동 재실행)
```bash
npm test -- --watch
```

### 5. 상세 결과 보기
```bash
npm test -- --verbose
```
